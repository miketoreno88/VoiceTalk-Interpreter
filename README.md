# VoiceTalk (EN -> RU)

Приложение захватывает входящий аудиопоток (микрофон или виртуальный loopback-источник с Zoom), распознает английскую речь и показывает перевод на русский в реальном времени.

## Что умеет

- Захват аудио с выбранного устройства.
- Автовыбор BlackHole, если устройство не указано.
- Распознавание английской речи через `faster-whisper`.
- Перевод текста на русский через Google Translate API (библиотека `deep-translator`).
- Окно с двумя лентами: оригинал и перевод.
- Сохранение субтитров в текстовый файл.
- Режим окна "поверх остальных" (удобно держать над Zoom).

## Установка

1. Установить Python 3.10+.
2. Установить зависимости:

```bash
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install -r requirements.txt
```

3. Установить BlackHole (нужен пароль администратора) и перезагрузить macOS:

```bash
brew install --cask blackhole-2ch
```

## Запуск

Быстрый запуск одной командой:

```bash
./run.sh
```

Проверка готовности окружения:

```bash
./run.sh --doctor
```

Показать доступные входные устройства:

```bash
python3 main.py --list-devices
```

Запустить с выбранным устройством:

```bash
python3 main.py --device 2
```

Можно выбрать другую модель Whisper:

```bash
python3 main.py --device 2 --model small.en
```

Запуск с сохранением в свой файл субтитров:

```bash
python3 main.py --subtitles-file ./my-subtitles.txt
```

## Как подключить Zoom-аудио на macOS

Для захвата звука из Zoom обычно нужен виртуальный аудиодрайвер (например, BlackHole):

1. Установить BlackHole и перезагрузить macOS.
2. В macOS Audio MIDI Setup создать Multi-Output/виртуальную схему.
3. В Zoom выбрать нужный выход.
4. Можно явно выбрать input-устройство BlackHole через `--device`, либо не указывать `--device` и дать приложению выбрать BlackHole автоматически.

Без loopback драйвера приложение слышит только микрофон.

## Ограничения

- Перевод зависит от интернет-доступа.
- Для very low-latency можно уменьшить `chunk_seconds` в `main.py`.
- Качество распознавания зависит от источника звука и выбранной модели.
